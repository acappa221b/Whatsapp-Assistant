# Launcher assets

Place `whatsapp-assistant.ico` here for a custom Windows shortcut icon.

## Windows shortcut

1. Duplo-clique em `Start WhatsApp Assistant.bat` (ou crie atalho na área de trabalho).
2. Clique direito no atalho → **Propriedades** → **Alterar ícone**.
3. Selecione `assets/whatsapp-assistant.ico`.

Optional: run `scripts/create-shortcut.ps1` when available to generate a `.lnk` automatically.
