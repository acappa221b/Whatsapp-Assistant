# Release RC-22A — Message-driven sync

**Versão:** 1.6.0-rc22 (MINOR)

## Novidades

- Chats descobertos somente por mensagem (default)
- Grupos e agenda não sincronizam automaticamente
- Toggles em Configurações → WhatsApp → Sincronização
- Permissões paginadas com filtros
- Botão **Limpar chats sem mensagem**
- Poll mais lento com aba inativa (Page Visibility)
