# Supported Formats

## Imagens suportadas

- `image/jpeg`
- `image/jpg`
- `image/png`
- `image/webp`

## Documentos suportados

- `application/pdf`

## Não suportados

Arquivos fora dessa lista resultam em:

- `UNKNOWN`
- evento `MediaUnsupported`
- nenhuma criação de entidade financeira

## Áudio

`AUDIO` continua `NOT_IMPLEMENTED` nesta Epic.

Implementação prevista para a Epic 11.
